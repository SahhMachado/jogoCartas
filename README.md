# JogoCartas
O jogo baseia-se em uma atividade prática realizada durante o curso técnico de informática. 
Consiste em uma espécie de jogo de cartas no qual o jogador com mais pontos (somatório das cartas) vence. Dessa forma, o usuário irá jogar contra o computador, escolhendo o naipe e número de cartas, 
sendo possível sorteá-las quantas vezes desejar antes de jogar. Após "jogar" as cartas, é realizado o somatório dos pontos do jogador e do computador, exibindo o vencedor.
